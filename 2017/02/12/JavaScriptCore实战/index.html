PersonJSExports 和 Person

Person 类实现了PersonJSExports 协议，该协议规定哪些属性在 JavaScript 中可用。

由于 JavaScriptCore 没有初始化，所以 create... 类方法是必要的，我们不能像原生的 JavaScript 类型那样简单地用 var person = new Person()。
